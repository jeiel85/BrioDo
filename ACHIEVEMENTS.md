## 구현 우선순위 제안

### Phase 1 — 이미 부분 구현 (데이터 있음, UI 추가만)
- S1 (🔥 streak 3일) ✅
- D1 (📥 인박스 제로) ✅
- C2 (⚡ 10개 완료) ✅

### Phase 2 — 기존 데이터로 바로 계산 가능
- S2, S3, S4, S5 (streak 확장)
- C3, C4, C5, C6 (완료 누적)
- D5, D6 (하루 완료 수)
- E1 (첫 업적 메타)

### Phase 3 — 새 데이터 필드 필요
- D2, D3 (시간대 기반) → `completedAt` 타임스탬프 필요
- R1~R4 (반복 일정) → 반복 기능 구현 후
- ST1~ST4 (하위 태스크) → 체크리스트 기능 구현 후
- AI1~AI4 → AI 입력 사용 카운터 필요

### Phase 4 — 특별/복합 조건
- SP1, SP2, SP3 (특별 날짜)
- P3 (균형 잡기)
- W3 (주말 집중)

## 기술 구현 노트

```javascript
// 업적 체크 함수 (useTodosData.js 또는 별도 useAchievements.js)
const ACHIEVEMENT_DEFS = [
  { id: 'S1', check: (stats) => stats.currentStreak >= 3 },
  { id: 'S2', check: (stats) => stats.currentStreak >= 7 },
  { id: 'C2', check: (stats) => stats.totalCompleted >= 10 },
  { id: 'C4', check: (stats) => stats.totalCompleted >= 100 },
  { id: 'D1', check: (stats) => stats.todayActive === 0 && stats.allTotal > 0 },
  // ...
]

// 업적 상태는 Firestore userSettings/{userId}/achievements 에 저장
// { achievementId: { unlockedAt: timestamp } }
```
