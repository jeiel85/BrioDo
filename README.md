# BlenDo (블렌두)

**BlenDo**는 단순한 체크리스트를 넘어, 당신의 일상과 기술을 하나로 부드럽게 섞어주는(Blend) 똑똑한 할 일 관리 앱입니다.

## ✨ 주요 특징 (Features)

1. **🧠 AI 스마트 입력**: "내일 오후 3시에 강남역 미팅"이라고 입력하면 AI가 날짜, 시간, 태그를 자동으로 추출하여 일정을 생성합니다.
2. **📱 직관적인 UI/UX**: 
   - 캘린더 형태의 부드러운 **주간 스와이프/무한 스크롤 뷰**
   - 전체 일정을 모아보는 **리스트 뷰**
   - 접었다 펼 수 있는 스마트 **태그 필터링**
3. **🎨 자유로운 커스터마이징 (테마 시스템)**:
   - 라이트, 다크, 시스템 기본 테마 지원
   - 🎲 **랜덤 테마**: 매번 새롭고 조화로운 HSL 컬러 팔레트가 화면 전체와 상태 표시줄(Status Bar)까지 완벽하게 스며듭니다.
   - 글자 크기(작게/중간/크게) 및 기본 보기 모드 설정.

## 🚀 앞으로의 로드맵 (Roadmap)

- **📶 로컬 퍼스트 & 오프라인 모드**: 인터넷이 없어도 즉시 일정을 추가하고 렌더링하도록 IndexedDB를 활용해 최적화합니다.
- **🔄 완벽한 동기화 & 충돌 관리**: 여러 기기에서 사용 시 수정 이력을 관리하고 디바이스 간 충돌을 방지합니다.
- **🗓️ 구글 캘린더 양방향 동기화**: BlenDo에 적은 일정이 내 구글 캘린더에도 즉각 반영되도록 지원합니다.
