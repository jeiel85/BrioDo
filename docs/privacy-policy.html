BrioDo 개인정보처리방침

최종 수정일: 2026년 3월 26일  |  시행일: 2026년 3월 26일

BrioDo는 사용자 개인정보를 소중히 여기며, 수집하는 정보를 최소화하고 앱 기능 제공 목적으로만 사용합니다. 광고 목적의 제3자 데이터 판매는 일절 없습니다.

1. 수집하는 정보

BrioDo는 다음 정보를 수집합니다.

  • Google 계정 정보: 이메일 주소, 표시 이름, 프로필 사진 (Google 로그인 시)
  • 할 일 데이터: 제목, 날짜/시간, 태그, 우선순위, 완료 여부, 반복 설정 등 사용자가 직접 입력한 정보
  • 앱 설정: 테마, 폰트 크기, 언어, 잠금화면 설정 등 사용자 환경 설정
  • AI 사용량: Gemini AI 기능 일일 사용 횟수 (기기 ID 기반, 개인 식별 불가)

2. 수집하지 않는 정보

  • 위치 정보
  • 연락처, 사진, 파일 등 기기 미디어
  • 마이크 (음성 인식은 Android 시스템에서 처리되며 BrioDo가 직접 녹음하지 않음)
  • 광고 식별자 (IDFA / GAID)
  • 결제 정보

3. 정보 이용 목적

  • 할 일 저장 및 기기 간 동기화 (Firebase Firestore)
  • Google 캘린더 연동 및 이벤트 동기화
  • AI 자연어 처리 (Gemini 2.5 Flash — 일일 10회 제한)
  • 로컬 알림 및 리마인더 전송
  • 앱 설정 저장 및 복원

4. 제3자 서비스

BrioDo는 다음 제3자 서비스를 사용하며, 각 서비스의 개인정보처리방침이 적용됩니다.

  • Google Firebase (Auth / Firestore): 인증 및 데이터 저장 — Firebase 개인정보처리방침
  • Google Calendar API: 캘린더 이벤트 동기화 (사용자 명시적 허가 후) — Google 개인정보처리방침
  • Google Gemini AI: 자연어 텍스트 분석 (할 일 입력 시에만) — Gemini 이용약관

5. 데이터 저장 및 보안

  • 로컬 저장: 기기 내 IndexedDB — 앱 삭제 시 자동 삭제
  • 클라우드 저장: Firebase Firestore — Google Cloud 인프라, AES-256 암호화
  • 접근 제어: Firestore 보안 규칙으로 본인 데이터만 접근 가능
  • BrioDo 개발자는 개별 사용자 데이터에 접근하지 않습니다.

6. 데이터 보관 및 삭제

  • 앱 내 계정 삭제 기능 또는 Firebase Console을 통해 모든 데이터를 직접 삭제할 수 있습니다.
  • 앱을 삭제하면 기기 내 로컬 데이터는 자동 삭제됩니다.
  • 클라우드 데이터 삭제 요청: 아래 이메일로 문의해 주세요.

7. 아동 개인정보 보호

BrioDo는 만 13세 미만 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

8. 방침 변경

개인정보처리방침이 변경될 경우, 이 페이지에 게시하여 공지합니다. 중요한 변경 사항은 앱 내 알림으로 안내할 수 있습니다.
